¿Quienes somos?

Estás en san-leon.com, un sitio web creado para mostrar un poco nuestro trabajo y conocernos. Somos un emprendimiento dedicado al desarrollo de aplicaciones web, siempre a medida con un funcionamiento sencillo pero potente. Nos ocupamos desde el diseño Front-end (lo que estas viendo ahora) hasta el Back-end (lo que no se ve, pero hace todo posible). Más abajo vas a ver algunos de nuestros últimos trabajos.

Fotografía

Un empujón más

Con un poco más de experiencia en sitios de fotografía, el tratamiento de los datos y lo que se necesita, encaramos este nuevo proyecto. Esta vez, a todo lo anterior había que mejorarlo y así surgió Ely Mar Fotografia, además de crear un sitio para exponer la gran trayectoria de nuestro cliente, lo dotamos con soporte para dos idiomas y todo el motor de gestión de datos que permite modificar a gusto las secciones. No dejen de visitarlo.
